# Delivery Delay Predictor using Logistic Regression

This project predicts whether a delivery will be **on time or delayed** using **Logistic Regression**.  
It is built with **Python, Pandas, scikit-learn, and Streamlit**, allowing users to interactively input delivery details and get predictions.

---

## 🚀 Features
- Predicts delivery delay based on order details and delivery information  
- Interactive **Streamlit app** for easy user input  
- Displays probability of delay along with prediction  
- Evaluates model using metrics like **Accuracy, Recall, F1-score, and ROC-AUC**  

---

## 📂 Dataset
- File: `delivery_delay.csv`  
- Features include: `Order ID, Customer, Product, Distance, Delivery Time, etc.`  
- Target variable: `Delivery Status` (0 = On Time, 1 = Delayed)
